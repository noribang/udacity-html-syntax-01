### *Part 2: Structure*
-Basic structure:
        ```
        <!DOCTYPE html>
        <html>
            <head>
                <meta charset="utf-8">
                <title>HTML Structure</title>
            </head>
            <body>
                <h1>This is a heading.</h1>
            </body>
        </html>
        ```
